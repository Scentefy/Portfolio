Games

Spirit Animal

A maze game based off a custom game made in Warcraft 3: The Frozen Throne game. The game is a puzzle timing game where you use your mask powers to overcome the obstacles.

Game was built in C++, with SDL Framework & Libraries, FMOD Libraries
and SDL_ttf Libraries.

Dodgey Ball

A 3D game based off dodge ball. This is a multiplayer game that requires 2 people and 2 xbox controllers to play

Game was built in C++, with Direct X Tool Kit, FMOD Libraries
and PhysX 3.2.4 Libraries

Note: This was a group project consisted of 3 people working on it for 7 weeks + upskilling
